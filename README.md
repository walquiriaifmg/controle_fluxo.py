# Estudos em Python

## Descrição
Programa em Python com menu para praticar estruturas condicionais, laços de repetição e comandos em git.

## Funcionalidades
1.Contar de 1 até um número usando for
2.Calcular média de notas usando while
3.Mostrar a tabuada de um número usando for
4.Sair do programa usando break

## Conteúdos praticados
- if, elif, else
- for
- while
- break
-Git init, add, commit e push
- Publicação de repositório no GitHub

## Como executar
- git config --global user.name "Seu Nome"
- git config --global user.email "seu-email@exemplo.com"
- git config --global --list
- crie o arquivo controle_fluxo.py
- git init
- git status
- git add controle_fluxo.py
- git status
- git commit -m "..."
- git log --oneline
